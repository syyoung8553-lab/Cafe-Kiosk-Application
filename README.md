## ✅ 각 화면병 영역 구분

① 메뉴 선택 영역

## AI 활용

1. AI를 통해 기본적인 구현 방향과 예시 코드를 확인한 뒤, 각 코드가 실행되는 시점과 화면에 반영되는 과정을 직접 확인하며 학습에 활용하기
2. 메뉴 선택, 장바구니에서 + / - 버튼을 통한 수량 추가·삭제 로직과 총 주문 금액 계산 과정에서 코드 흐름을 설명받으며 학습하기
