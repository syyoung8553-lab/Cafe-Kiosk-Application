

# Kiosk Application 만들기

실제 생활에서 사용하는 Kiosk UI를 더해, 실제 앱으로 완성해보기

## 💻 사용한 기술
- Swift Xcode

## 프로젝트 개요
 프로젝트 제목
카페 키오스크 앱 Always~Lovely.

 간단 설명
카페 키오스크 주문 앱으로, 메뉴 선택부터 장바구니 관리, 결제까지의 흐름을 구현한 
팀 프로젝트입니다.

## ✅ 각 화면별 영역 구분

① 메뉴 선택 영역
  - 카테고리(SegmentedControl)로 커피 / 스무디 / 디저트 분리
  - CollectionView를 사용해 2×2 메뉴 배치
  - 메뉴 선택 시 장바구니에 자동 추가

② 장바구니 영역
  - TableView로 주문 내역 표시
  - - / + 버튼으로 수량 조절
  - 수량 0이 되면 자동 삭제
  - 장바구니가 비어 있으면 안내 문구 표시

③ 결제 영역
  - 총 주문 금액 실시간 계산
  - 취소하기 버튼 → 확인 Alert 후 전체 삭제
  - 결제하기 버튼 → 결제 완료 Alert 표시


## AI 활용

1. 메뉴 이미지 생성에서 통일성을 위해서 생성형 AI를 활용했습니다.
2. 메뉴 카테고리, 장바구니, + / - 버튼으로 수량 추가·삭제, 총 주문 금액 계산 과정, 취소하기 결제하기 버튼과 누르면 뜨는 화면구현 AI를 활용해서 코드 작성 
   사용한 이유는  아직 UI Kit 코드 작성이 익숙하지 않아서 활용하였습니다.
3. AI를 안 쓴 부분에서 썼다면 안전했을지 AI가 준 코드를 이해할 수 있고 디버깅 오류를 수정할 수 있었을 거 같습니다.
